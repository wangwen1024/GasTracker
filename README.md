# GasTracker

基于Java构建的以太坊气体追踪器

Building an Ethereum Gas Tracker on Java

支持Alchemy,Infura及web3底层升级了EIP-1559提案的所有节点

Support Alchemy, Infura and web3 bottom layer upgrade all nodes of EIP-1559 proposal
